# 테스트 이미지 만들기 - 초간단 가이드

## 방법 1: Word 사용 (가장 추천) ⭐⭐⭐⭐⭐

### 단계별 설명:

1. **Microsoft Word 열기**
   - Word 프로그램 실행

2. **수식 입력**
   - 방법 A: `Alt + =` 키 누르기
   - 방법 B: 상단 메뉴 → 삽입 → 수식

3. **수식 입력 예시:**
   ```
   2 + 3
   ```
   또는
   ```
   10 - 5
   ```
   또는
   ```
   4 × 6
   ```

4. **스크린샷 찍기**
   - `Win + Shift + S` 키 누르기 (Windows 스크린샷)
   - 수식 부분만 드래그해서 선택
   - 자동으로 클립보드에 복사됨

5. **이미지로 저장**
   - 그림판 열기 (Windows 검색 → "그림판")
   - `Ctrl + V` (붙여넣기)
   - 파일 → 다른 이름으로 저장
   - 형식: PNG 선택
   - 저장!

## 방법 2: 그림판 사용 (더 간단) ⭐⭐⭐⭐

1. **그림판 열기**
   - Windows 검색 → "그림판"

2. **텍스트 입력**
   - 상단 메뉴에서 "A" (텍스트) 아이콘 클릭
   - 큰 글씨로 수식 입력: `2 + 3`
   - 글씨 크기: 최소 24pt 이상

3. **저장**
   - 파일 → 다른 이름으로 저장
   - PNG 형식으로 저장

## 방법 3: 온라인 도구 사용 ⭐⭐⭐

### 사이트 1: LaTeX 수식 이미지 생성
- **URL**: https://www.codecogs.com/latex/eqneditor.php
- **사용법**:
  1. 웹사이트 접속
  2. LaTeX 코드 입력 (예: `2+3`)
  3. 이미지 다운로드

### 사이트 2: 수식 편집기
- **URL**: https://www.mathcha.io/
- **사용법**:
  1. 웹사이트 접속
  2. 드래그 앤 드롭으로 수식 만들기
  3. 이미지로 내보내기

## 발표용 추천 수식 (난이도별)

### Level 1: 초간단 (100% 성공 예상)
```
2 + 3
10 - 5
4 × 6
8 ÷ 2
```

### Level 2: 중간 (90% 성공 예상)
```
2 + 3 × 4
(2 + 3) × 4
2³ + 3²
√16 + √9
```

### Level 3: 복잡 (70% 성공 예상)
```
sin(30°) + cos(60°)
log₁₀(100) = 2
|x - 5| = 3
```

## 팁

1. **발표용은 Level 1-2만 사용**
   - 실패 위험 최소화
   - 확실한 것만 보여주기

2. **글씨 크기**
   - 최소 24pt 이상
   - 명확하게 보이게

3. **배경**
   - 흰색 배경
   - 깨끗하게

4. **간격**
   - 숫자와 연산자 사이 적절한 간격
   - 너무 붙이지 않기
